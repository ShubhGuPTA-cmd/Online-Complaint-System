
🖥️ Online Complaint Management System
📌 Overview

The Online Complaint Management System is a web-based application that allows users to register complaints and track their status, while administrators manage and resolve them efficiently. This project demonstrates full-stack development by integrating frontend, backend, and a MySQL database.

🚀 Features
User Registration & Login
Complaint Submission
Real-time Complaint Tracking
Status Updates (Pending, In Progress, Resolved)
Admin Dashboard
Follow-up Management
Secure Database with Constraints
🏗️ Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Node.js / PHP / Java
Database: MySQL

-Copy path till the civicdesk and do cd path on cmd
